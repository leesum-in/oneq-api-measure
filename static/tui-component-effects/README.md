Effects
===============
Effect component<br>
Make element move, appear, disappear with options.

## Feature
* Make a element move with slide effect
* Changeable Each options
	* duration
	* effect
	* direction(slide)
	* movement(slide)
* Make a element fade in/out
* Support effects such as
	* linear
	* easeIn
	* easeOut
	* easeInOut
